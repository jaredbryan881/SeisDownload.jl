## Installation

1. `add https://github.com/kura-okubo/SeisDownload.jl`
2. `using Pkg; Pkg.build("SeisDownload")`
3. You can run examples in `EXAMPLE` directory: <br>
`cd SeisDownload/EXAMPLE/Noise_BP; julia exec.jl`
